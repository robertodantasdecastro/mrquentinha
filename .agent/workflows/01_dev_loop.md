# Workflow 01 - Dev Loop

1. Atualizar branch base (`git pull`) e revisar `git status`.
2. Escolher item do `TODO_NEXT` ou tarefa ativa.
3. Implementar em incrementos pequenos e revisaveis.
4. Validar localmente (tests/lint/build/smoke conforme escopo).
5. Atualizar `PROJECT_STATE`, `CHANGELOG` e `DECISIONS` quando houver impacto.
6. Commitar com mensagem objetiva e push.
