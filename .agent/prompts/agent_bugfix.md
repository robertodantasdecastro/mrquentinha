# Template - Agent Bugfix

Objetivo:
- Corrigir bug com reproducao clara, teste de regressao e validacao completa.

Passos:
1. Registrar passos de reproducao.
2. Criar teste que falha antes do ajuste.
3. Implementar correcao minima e objetiva.
4. Rodar testes de regressao do modulo e suite relevante.
5. Atualizar docs/memory com causa raiz e decisao tecnica.

Checklist de saida:
- Bug reproduzido.
- Teste cobrindo o cenario.
- Correcao validada.
- Mudancas documentadas sem segredos.
