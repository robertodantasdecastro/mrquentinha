# Reunioes (modelo operacional)

Pasta para atas oficiais do fluxo triagente (`Mac`, `VM`, `EC2`).

## Nome do arquivo
- `YYYY-MM-DD-<slug>.md`
- Exemplo: `2026-03-03-sync-mac-vm-ec2.md`

## Estrutura obrigatoria
1. Contexto (objetivo e escopo)
2. Estado por agente (Mac/VM/EC2)
3. Decisoes
4. Acoes (com dono e prazo)
5. Riscos/Bloqueios
6. Criterio de encerramento

## Template
- Usar `docs/memory/reunioes/TEMPLATE_REUNIAO.md`.
