- [ ] Código implementado conforme requisito
- [ ] Testes criados e passando
- [ ] Lint/format sem erros
- [ ] Migrações criadas/aplicadas quando necessário
- [ ] Endpoints protegidos por RBAC
- [ ] Documentação atualizada (CHANGELOG + DECISIONS/ADR)
- [ ] Como testar descrito no PR/commit
